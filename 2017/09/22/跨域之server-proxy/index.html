跨域之server proxy

Posted on 2017-09-22
Words count in article:

继《跨域之jsonp》和《跨域之CORS》之后，还有一种比较常用的方法：Server Proxy，顾名思义，服务器代理的意思；如果说前两种方法是前端直接和跨域目标服务器发送直接请求（无论是src还是ajax），那么服务器代理则是由本域服务器请求跨域服务器（服务端不存在同源策略，也就不存在跨不跨域这一说），然后将得到的信息发送给前端，这一过程中页面只是请求了本域服务器，并没有直接和跨域服务器发生直接关系，所以也很好理解这为什么叫服务器代理。
